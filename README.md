en este proyecto creamoa una pagina web en la cual mostraba un catalogo de telas de colores con el objetivo de simular una tienda de ropa de egresados. para hacer esto usamos elementos basicos como:

# 1. Carousel (Carrusel)
Es el componente estrella que simula las páginas del libro. Permite que el usuario pase de una gama de colores a otra (Azules, Rojos, etc.) de forma fluida.

# 2. Grid System (Sistema de Rejillas)
Utilicé las clases .row y .col-4 para organizar las muestras de tela. Esto permite que en cada página se vean 3 colores perfectamente alineados y proporcionales.

# 3. Cards (Tarjetas)
Se aplicó en la sección de contacto para envolver el formulario de calificación. Le da un aspecto de "recuadro" limpio con bordes suaves y sombra.

# 4. Buttons (Botones)
Usé diferentes estilos: .btn-kalla (personalizado), .btn-outline-secondary para volver al inicio y .btn-dark para resaltar la sección de contacto.

# 5. Forms (Formularios)
Utilicé el componente .form-control dentro de un textarea para que los alumnos y padres puedan escribir sus reseñas de forma cómoda y estética.

# 6. Typography (Tipografía)
Apliqué clases como .display-5 y .fw-light para que el nombre de la marca Kalla y los títulos de las páginas tengan un aire de revista de moda.

# 7. Spacing Utilities (Utilidades de Espacio)
Clases como .mb-5 (margen inferior) y .py-5 (relleno vertical) son fundamentales en Bootstrap para que los elementos no estén amontonados y el diseño "respire".

# 8. Shadows (Sombras)
La clase .shadow-lg se aplicó al contenedor del carrusel para elevarlo visualmente del fondo beige, creando el efecto de que el libro está apoyado sobre una mesa.

# 9. Containers (Contenedores)
El componente .container es el que mantiene todo el contenido centrado y protegido de los bordes de la pantalla, adaptándose automáticamente si se ve desde un celular o una computadora.

# 10. Navs & Anchors (Navegación por Anclas)
Utilicé el sistema de navegación por IDs (#inicio, #catalogo, #contacto) que, combinado con Bootstrap, permite que los botones te lleven a diferentes "pestañas" de la misma página sin recargar.
